## 项目简介

在信息过载的时代，我们致力于为您提供一方纯净、安全的记账空间。Piggy Bank 摒弃了繁琐的云端同步与广告干扰，回归财务管理的本质：记录与规划。所有数据均严格存储于您的设备本地，确保每一笔资金变动都分毫不差且绝对隐私。

## 核心特性

- **极致安全与隐私**：采用纯本地文件存储机制，无需联网权限，彻底杜绝数据泄露风险。您的财务数据完全掌握在您自己手中。
- **金融级高精度运算**：内置自主研发的高精度字符串运算引擎，从底层杜绝浮点数计算误差，确保长期记账的准确性。
- **多目标管理模式**：支持“累积型”与“目标型”双模式存钱罐，直观展示存款进度、差额及达成百分比，助您高效实现财富目标。
- **深度交互体验**：经过深度优化的手势操作体系，支持卡片滑动快捷存取、自定义快捷指令及丝滑的物理滚动反馈。
- **全局账单透视**：提供按日期折叠的全局流水视图，支持多维度财务统计，让您对收支状况一目了然。
